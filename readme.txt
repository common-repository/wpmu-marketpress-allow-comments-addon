== Description ==

A simple addon that will allow comments to be added to product listing.
When the addon is ON, you can select on which elements to add/remove comments.
BUT you will have to manually set comments to created listings!!!

But first thing you will need the [MarketPress plugin from WPMU](http://premium.wpmudev.org/project/e-commerce?ref=sirshurf-89644).

== Installation ==

1. Download
2. Upload to your `/wp-contents/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
